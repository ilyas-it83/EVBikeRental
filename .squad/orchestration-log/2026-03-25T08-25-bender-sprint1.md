## API Contracts

### Auth
- `POST /api/auth/register` → { id, email, name, isAdmin } + cookies
- `POST /api/auth/login` → { id, email, name, isAdmin } + cookies
- `POST /api/auth/refresh` → new tokens + cookies
- `POST /api/auth/logout` → cookies cleared
- `GET /api/auth/me` → { id, email, name, isAdmin }

### Stations
- `GET /api/stations?lat=X&lng=Y&radiusKm=Z` → { id, name, lat, lng, availableBikes, emptyDocks }
- `GET /api/stations/:id` → { id, name, lat, lng, address, bikes: [ { id, status, bikeNumber } ] }

## Technical Stack

- **DB:** SQLite via better-sqlite3, Drizzle ORM, WAL mode
- **Auth:** JWT (15min access, 30d refresh) in httpOnly cookies, bcrypt cost 12
- **API Pattern:** Controllers → Services → DB. Zod input validation. Consistent error format.
- **Middleware:** `requireAuth`, `requireAdmin`
